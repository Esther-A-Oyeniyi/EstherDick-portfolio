Gravity Model: The impact of Exchange rate Volatility and Agricultural Exports

This project was completed using SPSS, Eviews and Stata. Panel unit root test was performed to investigate the univariate characteristics of the variables. The model was estimated using the maximum likelihood estimator of the Stoichastic Frontier Gravity Model (SFGM). Alternatively, Fixed Effect (FE), Ordinary Least Square (OLS), the PPML and the Heckman model was used to confirm the robustness of the results obtained from the SFGM. Based on the Hausman test, the FE model is preferred to the random effects model

The impact of Sustainability Campaign
on Consumer Purchase Intention

The data was collected using an online survey targetted at consumers between 18 and 30 years. Multiple Linear Regression and Correlation analysis. Reliability testing with cronbach alpha was performed. The regression analysis was done including latent values of four independent variables (awareness of sustainability issues, attitude, credibility and trust in sustainability information, sustainability information through digital marketing, subjective norm) and purchasing intent as the dependent variable for regression analysis in order to estimate the model. All the analysis were done using SPSS.
