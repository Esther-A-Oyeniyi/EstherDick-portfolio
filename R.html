Bike-Share Project

Bike-share Project was geared towards solving the problem of increasing churn rate and proving insight on how to develop successful and efficient campaign strategy. The business question that this analysis sought to answer was, How do members (annual subscibers) differ from casual riders? The data cleaning and analysis was done using R programming language while Tableau was used for the visualization.
